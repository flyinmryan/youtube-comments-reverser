Clone this repository or download as zip

Unzip if necessary

Locate start.js and open in a text editor

You will need a YouTube Data API key. You can get one here:
https://console.developers.google.com/apis/credentials

After creating a new API key, search for "YouTube Data API" in the search bar

Click on the top search result.

Enable the API by clicking the "Enable" button.

Press the back arrow to return to the credentials page where you created the API key.

Select the API key to edit.

Towards the bottom of the page, check the checkbox next to the newly enabled YouTube Data API V3 in the dropdown list.

Click save.

Click "View API key" by selecting the three dots icon on the right side of the API key table.  Copy the key.

Return to the text editory with start.js open.  Paste it between the quotes on line 1.  The API key should replace the text YOUTUBE_DATA_API_KEY, but keep the quotes.

Go to chrome://extensions

In the top right, toggle the "Developer mode" ON

A few buttons will appear, select "Load Unpacked"

Select the folder containing the contents of this repository
